# # 🚀 Shadowrocket Strict Whitelist Lists

Сборка кастомных конфигураций для Shadowrocket, построенная по принципу **Strict Whitelist** (Всё в прокси, кроме проверенных российских сервисов напрямую).

### 📦 Что внутри (Merged):
* **roscomvpn** — базовые списки обхода
* **hxehex & VasiliyBP** — домены и IP инфраструктуры
* **misha-tgshv** — банковские домены и финтех-сервисы
* **community** — кастомные фиксы от сообщества

### 🛠 Логика работы:
Вся гео-блокировка и заблокированные ресурсы улетают в `PROXY` через правило `FINAL`, а критически важные RU-сервисы идут напрямую через `DIRECT`.

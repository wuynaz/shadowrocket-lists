# 🚀 Shadowrocket Strict Whitelist

Единый whitelist для **Shadowrocket**, собранный из нескольких открытых проектов и объединённый в один актуальный список.

Конфигурация построена по принципу **Strict Whitelist** — всё неизвестное отправляется через **PROXY**, а белые списки работают напрямую (**DIRECT**).

## 📦 Источники

Whitelist автоматически собран из нескольких публичных репозиториев:

* **hxehex/russia-mobile-internet-whitelist**
* **Sereuser/russian-whitelists**
* **jsxta/whitelist-russia**
* **d010b/custom_geoip**
* **chebur-net/russia-mobile-whitelist**
* **escapingworm/russia-whitelist**
* **kort0881/russia-whitelist**

Все найденные IP-адреса, CIDR-подсети и домены объединяются, очищаются от дубликатов и используются для формирования единого списка.

## 🛠 Логика работы

* 🇷🇺 Российские сервисы из whitelist → **DIRECT**
* 🏠 Локальная сеть → **DIRECT**
* 🚫 Реклама, трекеры и телеметрия → **REJECT**
* 🌍 Остальной интернет → **PROXY**

## ⚡ Особенности

* Объединение нескольких независимых источников.
* Очистка и дедупликация IP/CIDR.
* Поддержка более **30 000** сетей и **1000+** доменов.
* Конфигурация полностью готова к импорту в Shadowrocket.
* Регулярное обновление списка по мере появления изменений в исходных репозиториях.
